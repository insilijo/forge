# Report an Issue

Found a bug or want to request a feature? Use the forge issue tracker on GitHub.

[Open an issue](https://github.com/insilijo/forge/issues/new/choose){ .md-button .md-button--primary }

---

## Before you file

- Check the [FAQ](faq.md) — common problems are documented there
- Search [existing issues](https://github.com/insilijo/forge/issues) to avoid duplicates

## Bug reports

Please include:

- What you did (steps to reproduce)
- What you expected to happen
- What actually happened
- Browser and OS if it's a UI issue
- Any error messages shown on screen

## Feature requests

Describe the use case and what outcome you're trying to achieve. If there's a workaround you're currently using, mention it.

## Which component

If you know which part of the platform is involved, note it in the title:
